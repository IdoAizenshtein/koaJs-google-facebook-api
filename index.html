About our product

We allow users to manage their "Google tasks" through WhatsApp messages. First you have to register, in the registration process the user is identified using his WhatsApp number as the only identifier for the account.
After that, the user will receive a link by Google for authentication and permission to manage "Google Tasks".
After that, it will be possible to receive on WhatsApp the "Google task" lists, live updates on tasks with an upcoming date, the ability to edit and delete, and more - all this directly through WhatsApp only!
